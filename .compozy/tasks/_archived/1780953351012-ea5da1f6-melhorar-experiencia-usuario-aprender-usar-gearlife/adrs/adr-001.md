# ADR 001: Update the info modal and link it from the existing How It Works page

Date: 2026-05-30
Status: Accepted

## Context

GearLife already includes an info modal accessible from the Header and Sidebar and a standalone `/como-funciona` page. The current modal content and images are outdated, and the user wants a self-serve learning experience that eliminates confusion.

## Decision

Keep the current info modal as the quick-start summary, update its copy and images to reflect the current app state, and add or improve a link from the existing `/como-funciona` page so users can access the full reference from the modal or navigation.

## Alternatives Considered

- **Convert the modal into the primary How It Works page**: simpler source of truth, but removes the fast-access quick-start experience and requires users to leave the current app flow.
- **Keep the modal for quick tips and build a richer reference page**: best coverage, but adds more content work and increases the risk of inconsistent messaging unless carefully synchronized.

## Consequences

- The app retains an instant-access help experience while the dedicated reference page remains available.
- The modal and the page must be kept synchronized, so content updates must consider both presentation contexts.
- Users can learn the app on their own without needing a separate onboarding flow or support intervention.
